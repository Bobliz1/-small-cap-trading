# 小市值策略内容

每日：

开盘前：过滤停牌股票后，获得市值百分之五到百分之十的前十位股票。

开盘时：在非调仓日，根据清仓信号添加清仓股票栏，清出清仓股票栏中的股票；在调仓日：先清出清仓股票栏中的股票后，再将股票数量补充至10只。

开盘后：更新最大净值的点，便于在下一个交易日计算清仓信号。
